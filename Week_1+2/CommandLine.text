Lệnh ls:
-a: Hiển thị cả các tệp ẩn.
-l: Hiển thị thông tin chi tiết về tệp và thư mục.
-h: Hiển thị kích thước dễ đọc (ví dụ: KB, MB).

Lệnh cd:
cd directory: Di chuyển đến thư mục directory.
cd ..: Di chuyển lên một thư mục cha.


Lệnh mkdir:
-p: Tạo cả thư mục cha nếu chưa tồn tại.


Lệnh cp:
-r hoặc -R: Sao chép thư mục đệ quy.
-i: Xác nhận trước khi ghi đè lên tệp.

Lệnh mv:
mv file1 file2: Đổi tên tệp từ file1 thành file2.
-i: Xác nhận trước khi ghi đè.

Lệnh rm:
-r hoặc -R: Xoá thư mục và nội dung đệ quy.
-f: Xoá mà không xác nhận.

Lệnh grep:
-i: Tìm kiếm không phân biệt chữ hoa và chữ thường.
-r: Tìm kiếm đệ quy trong tất cả các tệp và thư mục con.

Lệnh ps:
-e: Hiển thị danh sách tiến trình.
aux: Hiển thị tất cả tiến trình và thông tin chi tiết.

Lệnh ssh:
-p: Xác định cổng kết nối SSH.

Lệnh tar:
-cvzf: Nén thư mục thành tệp tar với gzip.
-xvf: Giải nén tệp tar.


Lệnh gzip : nén fille .gzip
Lệnh bzip2: nén file .bz2
Lệnh xz : nén file .xz
- Giải nén thì thêm tham số -d
Lệnh gzip : giải nén file .gz
Lệnh bz2: giải nén file .bz2
Lệnh xz :giải nén file .xz


Lệnh ifconfig:
-a: Hiển thị tất cả các giao diện, bao gồm cả giao diện ẩn.

Lệnh ping:
-c: Xác định số gói tin để gửi.

Lệnh wget:
-O: Đặt tên cho tệp tải về.

Lệnh sed:
-i: Thay đổi trực tiếp trên tệp gốc.
-e: Thêm biểu thức chính quy.
-g: thay đổi tất cả

Lệnh awk:
-F: Xác định ký tự phân cách trường.

Lệnh find:
-name: Tìm theo tên tệp hoặc thư mục.


lệnh uname : cung cấp thông tin hệ thống


Lệnh awk  : xử lí văn bản theo cột
- l: đếm số dòng
- w: đếm số từ
- c: đếm số ký tự
- m: đếm số ký tự đặt biệt và dấu cách
Lệnh wc : đếm từ, dòng, ký tự


Lệnh uniq : loại bỏ các dòng trùng lặp
- d: hiển thị các dòng bị lặp lại
- i: loại bỏ chữ hoa và thường
- c: hiển thị số dòng đã xuất hiện


Lệnh chmod (quyền) file
Lệnh cat /etc/passwd : xem tt người dùng
Lệnh cat /etc/group : xem tt nhóm 

Lệnh groups : xem tt nhóm của người dùng
Lệnh members: xem thành viên của nhóm

Lệnh useradd : thêm mới người dùng 
- c : thêm tt người dùng
- m : tạo thư mục home và gán người dùng
- p : yêu cầu đổi mật khẩu ngay lần đầu
Lệnh userdel : xóa người dùng
- r : xóa thư mục người dùng

Lệnh groupadd : thêm nhóm người dùng 
- g : thêm id cụ thể
Lệnh groupdel : xóa nhóm người dùng

Lệnh gpasswd: quản lí tt thành viên trong nhóm
- d : xóa người dùng khỏi nhóm
- a : thêm người dùng vào nhóm
- A : chỉ định người dùng làm quản lí

Lệnh xóa file : rm 
Lệnh xóa thư mục : rm -r
